ERENO

ERENO Traffic Generator.

ERENO is a tool that allows the generation of datasets with messages from the GOOSE and SV protocols for electrical substations. The existing modules enable you to configure parameters and define attacks, currents, and voltages. Finally, the resulting dataset is made available for download.

ERENO IEC-61850 Intrusion Dataset

The ERENO was set up to generate a suite containing 7 datasets, each covering one of the proposed attack use cases. Each dataset has 69 features (columns) and different numbers of attack and normal samples (rows). Information about each dataset and a complete description of the dataset can be found in this Ph.D. thesis.
